  Python Banking System (OOP)
A modular, multi-file banking application built with Python to demonstrate Object-Oriented Programming (OOP) and logic-driven software design.

  Project Overview
This project simulates a digital banking environment where users can securely create accounts, manage their credentials, and perform financial transactions.

  Key Features:
User Authentication: Secure signup and login system with unique 4-digit ID verification.
Transaction Logic: Functional deposit, withdrawal, and balance inquiry systems with error handling for numeric inputs.
Modular Architecture: Organized into separate files for better maintainability and scalability.

  Technical Stack
Language: Python 3.x
Concepts: Classes and Objects, Method Decorators (@staticmethod), and Exception Handling.
Database Integration: Permanent SQLite storage will be added as soon as possible.

 How to Run
To run the banking system, you must execute the program from the Main.py file:

Ensure all files are in the same folder:
Main.py
Signupfile.py
Loginfile.py

Run the entry point:
python Main.py
